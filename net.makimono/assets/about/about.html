JMDict

The JMDict project is a compilation of a multilingual Japanese dictionary XML-format database.

The JMDict content is used in the dictionary section of this application. All dictionary translations from Japanese to English / German / French / Russian are based on the JMDict.

KANJIDIC2

The KANJIDIC2 project is a consolidated XML-format kanji database combining the information currently in the KANJIDIC and KANJD212 files

The kanji section of Makimono uses the KANJIDIC2 content.

© ELECTRONIC DICTIONARY RESEARCH AND DEVELOPMENT GROUP
License: Creative Commons Attribution-ShareAlike 3.0
Website: www.csse.monash.edu.au

KanjiVG

KanjiVG is a description of the sinographs (or kanji) used by the Japanese language.

The kanji stroke animation and drawing instructions in Makimono use the SVG paths from the KanjiVG project.
